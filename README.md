# Gourmet Daily Website

全球美食日报网站 - 追踪四大洲美食情报

## 网站地址

🌐 https://gourmetdaily.vercel.app/

## 关于

Gourmet Daily 每日自动抓取美洲、欧洲、非洲、亚洲四大洲的美食资讯，涵盖：

- 🥖 **面包烘焙** - 各国传统与现代烘焙技艺
- 🧁 **精致甜点** - 法式甜点、日式和果子、意式甜品等
- 🍲 **风土人情** - 当地传统美食与文化故事

## 自动更新

网站每日自动更新两次（北京时间 8:00 和 20:00）

## 技术栈

- HTML5 + CSS3
- 微信分享优化
- Vercel 部署
